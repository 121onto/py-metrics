PyMetrics
=========

Use python for econometrics.  The API generally follows Stata, is built around the
Pandas project, and support modern econometric best practices.

Getting Started
---------------



Developer Installation
----------------------

Note: this repo is compatible with python 3.6 only.

1. Clone this repository:

    ```bash
    git clone git@github.com:121onto/py-metrics.git
    cd py-metrics
    ```

2. Create a new virtualenv (assumes you're using virtualenvwrapper):

    ``` bash
    mkv py-metrics --python=python3
    actv
    ```

3. Install the repository in editable mode:

    ```bash
    pip install -e .
    ```
